# StudyLog - Sistema de Gerenciamento de Sessões de Estudo

## 📋 Sobre o Projeto

Este projeto foi desenvolvido como **Checkpoint 2** do segundo semestre do curso de **Análise e Desenvolvimento de Sistemas** da **FIAP** - Turma 1TDSPX.

O **StudyLog** é uma aplicação web desenvolvida em React com TypeScript para gerenciar sessões de estudo. Os usuários podem registrar suas sessões de estudo, visualizar o histórico e acompanhar seu progresso acadêmico.

## 🚀 Funcionalidades

- **✅ Adicionar novas sessões de estudo**
- **📊 Visualizar lista de sessões cadastradas**
- **🔍 Ver detalhes completos de cada sessão**
- **🗑️ Remover sessões**
- **⏱️ Acompanhar tempo total de estudo**

## 🛠️ Tecnologias Utilizadas

- **React** - Biblioteca principal
- **TypeScript** - Tipagem estática
- **Tailwind CSS** - Estilização
- **React Router DOM** - Roteamento
- **React Error Boundary** - Tratamento de erros
